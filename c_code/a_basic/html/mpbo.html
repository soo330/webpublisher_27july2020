margin과 padding의 쓰임새를 조금 더 파악해보자

하나의 요소 내에서 첫번 째 존재하는 요소가면
float, position 처리되어 있지 않다면
margin-top값을 사용할 경우에는 자신이 아닌
부모요소를 margin처리하게된다.

margin값이 위아래 겹칠경우에는 큰 값 쪽만 처리가 된다
단, float처리된 겹침은 제외 (좌/우 해당x)

margin값은 눈에보이는 공간의 거리를 조절하는 기능이기에 + 값이 아닌 -값으로 처리된다

단, margin값은 auto, 좌위의 한해서 %값 처리가 가능하지만, 위/아래는 %로 사용하는 것은 지양
위/아래는 명확한 값을 파악하기가 어렵다 (부모의 높이를 기준으로 보는것이 아니다)

float기능과 margin기능을 같이 사용할 경우 margin:auto;는 좌/우 가운데 배치되지 않는다
